Overhauled stateless-svc NetworkPolicy and Ingress to remove explicit `enabled` flags and replace them with auto-detection from configured values.

- `stateless-svc/templates/networkpolicy.yaml`: NP rendered only when inbound or outbound rules are configured; `podSelector` fixed to target chart's own pods; `Egress` only added to `policyTypes` when egress rules exist; added `allowSameNamespace` flag on both sides
- `stateless-svc/templates/ingress.yaml`: gate switched from `if .Values.ingress.enabled` to `if .Values.ingress.hosts`
- `stateless-svc/templates/NOTES.txt`: updated post-install URL display to match gate change
- `stateless-svc/values.yaml`: removed `ingress.enabled`; `ingress.hosts` defaults to `[]`; added `allowSameNamespace` on ingress and egress
